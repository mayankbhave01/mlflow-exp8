# Experiment 8 - MLflow Model Registry

## Objective
To train and register a machine learning model using MLflow with SQLite backend.

## Steps Performed
1. Installed MLflow and dependencies
2. Loaded Iris dataset
3. Trained Random Forest model
4. Logged metrics (accuracy)
5. Registered model in MLflow
6. Viewed results in MLflow UI

## Output
- Model: IrisModel
- Accuracy: ~0.96
- Version: v1, v2

## Conclusion
MLflow was successfully used for model tracking and registry.
